# Shared Skills

## Core Skills For All Agents

- Translate product requirements into implementation-ready tasks
- Follow the defined tech stack strictly:
  - Backend: NestJS
  - Frontend: Next.js
  - Styling: Tailwind CSS
  - Database: MongoDB
  - Language: TypeScript
- Apply the DRY principle
- Use TypeScript types for everything
- Keep outputs concise
- Only provide code blocks unless explanation is requested

## Frontend Skills

- Build responsive UI with Next.js and Tailwind CSS
- Create functional React components only
- Define typed props, view models, and API consumption patterns
- Build reusable components and feature modules
- Implement accessible and user-focused interfaces

## Backend Skills

- Structure NestJS modules cleanly
- Design DTOs, services, and controllers with strong typing
- Model MongoDB entities and validation rules
- Create maintainable domain logic and API layers
- Build scalable backend foundations

## QA Skills

- Derive test cases from requirements
- Validate core flows and regressions
- Identify mismatched behavior across UI, API, and data layers
- Report concise and reproducible issues

## Integration Skills

- Align frontend and backend contracts
- Validate shared assumptions and field mappings
- Support environment and flow wiring
- Detect end-to-end breakpoints quickly

## Sprint Planning Skills

- Convert product requirements into phased delivery plans
- Prioritize using MoSCoW
- Sequence work by dependencies
- Assign clear ownership across agents
